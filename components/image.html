Images

Responsive images

Images in Bootstrap 3 can be made responsive-friendly via the addition of mw-100, h-auto, db classes that apply max-width: 100%;, height: auto; and display: block; to the image so that it scales nicely to the parent element.

To center images which use these classes, use .center instead of .tc. See the helper classes section for more details about .center usage.

Image shapes

EXAMPLE

<img src="..." alt="..." class="br3 ma1">
<img src="..." alt="..." class="br-100 ma1">
<img src="..." alt="..." class="pa1 ba b--black-20 br3 ma1">
